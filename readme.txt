== Description ==

A very simple way for adding Pin-it capability to your WP site. Just use [wmpin] [/wmpin]. 

It uses jquery for the Pinterest API. It is intended for one use and that is to make pinning easy.

== Installation ==

1. Upload `wm-pinterest` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode [wmpin] the image to pin[/wmpin]
